Dataset¶

This online retail dataset is from the UC Irvine Machine Learning Repository: https://archive.ics.uci.edu/dataset/502/online+retail+ii.

Our goal is to estimate customers' gross spend in the next one year as an effort to understand customer lifetime value.

First, we will load the data and display some basic metadata.

def num_missing_by_columns(df):
    num_missing = df.isnull().sum().sort_values(ascending=False)
    percent = (df.isnull().sum()/len(df)).sort_values(ascending=False)
    missing_data = pd.concat([num_missing, percent], axis=1, keys=['Missing', 'Percent'])
    return missing_data

Many of these are expenses. Our goal is to measure customers' gross spends so it should be fine to remove these. For simplicity, we will use some heuristics to do the filtering - we want the stock code to be between 5 and 8 characters, and include numbers.

In [15]:
def include_stock_code(code):
    if len(code) < 5 or len(code) > 8:
        return False

    if 'TEST' in code:
        return False

    for c in code:
        if c.isnumeric():
            return True

    return False

df_explore = df_explore[df_explore.apply(lambda x: include_stock_code(str(x['StockCode'])), axis=1)]

Let's dig into the data and ask some questions. In order to calculate the customer value in future periods, we first need to know if the customers are returning at all. We will include only purchases for the analyses below.

Returning customers¶

Cohort Analysis¶

In particular, we will use customers from the first 3 months as our cohort, and examine their retention rate.

In [632]:
cohort_cutoff = '2010-03-01'
cohort = df_eda.loc[df_eda['InvoiceDate'] < pd.to_datetime(cohort_cutoff), ['Customer ID']].drop_duplicates()
In [633]:
df_cohort = pd.merge(df_eda[df_eda['InvoiceDate'] >= pd.to_datetime(cohort_cutoff)], cohort, on='Customer ID', how='inner')
df_cohort['spend'] = df_cohort['Quantity'] * df_cohort['Price']
In [663]:
cohort_analysis = df_cohort.groupby(['YearMonth'])\
    .apply(lambda x: pd.Series({
        'num_customers': x['Customer ID'].nunique(),
        'num_invoices': x['Invoice'].nunique(),
        'avg_spend_per_invoice': x['spend'].sum() / x['Invoice'].nunique(),
    }))\
    .reset_index()

cohort_analysis['percent'] = cohort_analysis['num_customers']/cohort.shape[0]
cohort_analysis['frequency_per_customer'] = cohort_analysis['num_invoices']/cohort_analysis['num_customers']
cohort_analysis['YM'] = cohort_analysis['YearMonth'].astype(str)

Understandably, retention rate trended down as time goes on. It seems to stablizes around 20-25% at the end of a year. Average spend and number of orders on the other hand have a flat trend line, around 500 and 1.62 respectively. It is noteworthy that the spike around October and November around all three metrics. Since we only have a year of data here, we cannot confirm whether it is a seasonality/holiday effect, or a one-time occurrence, for example, a market campaign.
